# よくある質問

??? question "選考にはどれくらい時間がかかりますか？"
    応募からオファーまで、通常 2〜4週間 を目安にしています。
    ご事情に応じて柔軟に対応しますので、お気軽にご相談ください。

??? question "未経験でも応募できますか？"
    ポジションによって異なります。各求人の応募要件をご確認ください。

??? question "カジュアル面談は選考に含まれますか？"
    いいえ。カジュアル面談は選考とは別の場です。
    お互いの理解を深める目的で実施しています。

??? question "リモートワークは可能ですか？"
    [働き方](../culture/workstyle.md) ページをご確認ください。

??? question "面接の結果はどれくらいで届きますか？"
    原則として **5営業日以内** にご連絡します。
